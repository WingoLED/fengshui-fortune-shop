# Feng Shui Fortune Shop

A complete e-commerce and CMS website for Feng Shui products and services.

## Features

- **Role-based Management**: Admin, Owner, Editor, Subscriber
- **Product Management**: Add, edit, delete products with inventory
- **Content Management**: Blog-style tips system with daily tips
- **User Management**: Registration, login, favorites
- **Appointments**: Booking system for consultations
- **Responsive Design**: Works on desktop and mobile
